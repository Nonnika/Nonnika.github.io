Http Pattern模式

Pattern

什么是 Pattern ？

模式（pattern）是一个与 http请求匹配的 URL路径字符串。ServeMux使用模式来分配合适的处理函数。

规则和定义

  • 固定 URL 路径

    如：/hello 则只会处理 /hello 不会处理 /hello/

  • 子树路径

    如：/hello/ 会处理 /hello/hello.css

  • 最长匹配优先

    如：/hello/ 请求会匹配到 /hello/ 而不是 /

  • 特定主机模式

    当然你可以根据 host 不同来设定不同的处理函数如：www.hello.com/
